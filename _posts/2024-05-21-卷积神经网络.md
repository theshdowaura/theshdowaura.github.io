## 卷积神经网络



什么是卷积神经网络?

基本概念：

$$ \int _ { - \infty } ^ { \infty } f ( \tau ) g ( x - \tau ) d \tau$$

什么是卷积？卷积是一个泛函的概念。

现实公设，假设一个一天的进食情况，不仅吃一日三餐而且吃零食。

用$f$函数表示进食，用$g$函数表示消化

我们想知道这个在下午两点肚子里还未消化的食物还剩多少？
