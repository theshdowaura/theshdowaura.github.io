当
$$
x \to 0
$$
时,则为
$$
\sin x \sim x
$$

$$
\tan x \sim x
$$

$$
\ln ( 1 + x ) \sim x  
$$

$$
e ^ { x } - 1 \sim x
$$
$$
\arcsin x \sim x
$$
$$
\arctan x \sim x 
$$
$$
\log _ { a } ( 1 + x ) \sim \frac { x } { \ln x }
$$
$$
a ^ { x } - 1 \sim \ln a \cdot x
$$
$$
1 - \cos x \sim \frac { 1 } { 2 } x ^ { 2 }
$$
$$
( 1 + x ) ^ { n } - 1 \sim \frac { x } { n }
$$
